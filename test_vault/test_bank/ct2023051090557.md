# Q
低钾性碱中毒的可能发生机制是
A. 细胞外K+内移，肾近曲小管K+-Na+交换↓
B. 细胞外K+内移，肾近曲小管H+-Na+交换↑
C. 细胞外K+内移，肾近曲小管H+-Na+交换↓
D. 细胞内K+释出，肾近曲小管H+-Na+交换↑
E. 细胞内K+释出，肾近曲小管H+-Na+交换↓



D





# D
应注意，**钾紊乱导致的酸碱中毒**与**酸碱中毒导致的钾紊乱**是两码事

此题中，低血钾是原发病，导致了Na-H交换增强引起碱中毒为一级继发病，钾离子从胞内移向胞外为原发病的代偿表现
